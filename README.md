🌐 Thee Site

Welcome to Thee Site — a modern, visually engaging website built using HTML, CSS, and JavaScript.

🔗 Live Website: https://jas0n0-0.github.io/thee_site/

📌 About the Project

This project is a personal website designed to showcase creative web design skills, including layout structuring, styling, and interactive elements.

It features:

A clean and simple navigation bar
A full-screen video background
Styled sections for content display
Responsive design principles (in progress)
🛠️ Built With
HTML5 – Structure of the website
CSS3 – Styling, layout, and animations
JavaScript – Interactivity
GitHub Pages – Deployment and hosting
🎯 Features
🎥 Background video for a modern look
🎨 Custom styling and hover effects
📱 Responsive design (improving over time)
⚡ Fast and lightweight
🚀 Getting Started

To run this project locally:

Clone the repository:

git clone https://github.com/jas0n0-0/thee_site.git

Open the project folder:

cd thee_site
Open index.html in your browser.
